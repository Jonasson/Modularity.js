Attribute values

  • You should not have to enter an empty value ({}).
  • JSON is used today, but maybe a more terse one could be used. Knockout could be a source of inspiration in how it handles attribute values.

Module IDs

  • You should not have to give every module element an ID. The module key should be generated with a counter.
